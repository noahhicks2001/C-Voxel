
# C-Voxel
<pre>
WIP voxel rendering engine created in C using SDL2


CONTROLS: 
    w-> move up 
    a-> move left 
    s-> move down  
    d-> move right 
    u,j -> rotation around x-axis 
    h,k -> rotation around z-axis 
    y,i -> rotation around y-axis 
    r -> scale down  
    t -> scale up  
    arrow_left,arrow_right -> change selected voxel
  </pre>
